Create a responsive, low-fidelity wireframe for a modern real estate website
that will be developed using Next.js (App Router).
The wireframe should be component-based, modular, and mobile-first.

Use grayscale only (no colors, no real images).
Use rectangles, text placeholders, and simple icons.
Clearly indicate reusable components and responsive behavior.

====================================
GLOBAL TECH CONTEXT
====================================
Framework: Next.js
Layout style: Component-based
Responsive breakpoints:
- Desktop (≥1024px)
- Tablet (768–1023px)
- Mobile (≤767px)

====================================
PAGE: HOME PAGE (Landing Page)
====================================

------------------------------------
GLOBAL LAYOUT STRUCTURE
------------------------------------
- Root layout container
- Max-width content wrapper
- 12-column grid for desktop
- Single-column layout for mobile
- Adequate spacing and padding for touch devices

------------------------------------
HEADER / NAVBAR (Reusable Component)
------------------------------------
Desktop:
- Left: Logo placeholder
- Center/Right: Navigation menu
  - Properties (dropdown)
    - Buy
    - Rent
    - Commercial
    - New Projects
  - Interiors
  - About Us
  - Contact Us
- Right: Primary CTA button “Book a Call”
- Sticky header behavior

Mobile:
- Left: Logo
- Right: Hamburger menu icon
- Slide-in mobile menu containing:
  - Properties (accordion dropdown)
  - Interiors
  - About Us
  - Contact Us
  - CTA button

------------------------------------
HERO SECTION (Responsive Component)
------------------------------------
Desktop Layout:
- Two-column layout
  - Left column:
    - Large headline text:
      “Find a place you will call home”
    - Supporting description text
    - Primary CTA: “Explore Properties”
    - Secondary CTA: “Contact Us”
  - Right column:
    - Large image placeholder (luxury property)

Mobile Layout:
- Single-column layout
- Text stacked above image
- Center-aligned CTAs
- Reduced headline size

------------------------------------
PROPERTY SEARCH BAR (Reusable Component)
------------------------------------
Desktop:
- Horizontal search bar with:
  - Location (input)
  - Property Type (dropdown)
  - Budget (dropdown)
  - Search button

Mobile:
- Vertical stacked inputs
- Full-width search button
- Touch-friendly spacing

------------------------------------
FEATURED PROPERTIES SECTION
------------------------------------
Desktop:
- Section heading: “Featured Properties”
- Grid layout (3 cards per row)
- Property Card Component includes:
  - Image placeholder
  - Property title
  - Location
  - Price
  - “View Details” button

Mobile:
- Single-column card layout
- Horizontal scroll or stacked cards

------------------------------------
INTERIORS SECTION
------------------------------------
Desktop:
- Two-column layout
  - Left: Text content
  - Right: Image placeholder
- CTA button: “Explore Interiors”

Mobile:
- Stacked layout
- Text first, image below

------------------------------------
WHY CHOOSE US SECTION
------------------------------------
Desktop:
- Section heading
- 4 feature blocks in grid:
  - Icon placeholder
  - Title
  - Short description

Mobile:
- Vertical stacked feature blocks

------------------------------------
ABOUT US PREVIEW SECTION
------------------------------------
Desktop:
- Text block + image placeholder side-by-side
- “Read More” button

Mobile:
- Stacked layout

------------------------------------
CHATBOT (GLOBAL COMPONENT)
------------------------------------
- Floating chatbot icon fixed at bottom-right
- Label: “Chat with us”
- Expandable chat panel indicated
- Works across all screen sizes

------------------------------------
CONTACT CTA SECTION
------------------------------------
- Full-width section
- Text: “Looking for your dream property?”
- Primary CTA button: “Contact Us”

------------------------------------
FOOTER (Reusable Component)
------------------------------------
Desktop:
- Multi-column layout:
  - Logo
  - Quick Links
  - Contact Info
  - Social icons

Mobile:
- Stacked sections
- Center-aligned content

INTERACTION & UX NOTES
